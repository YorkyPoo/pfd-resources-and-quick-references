# Poquoson Fire and Rescue Patient Care Protocols
## Administrative - Abuse Recognition & Mandated Reporting

### Overview
EMS providers act as mandated reporters under state law for suspected elder abuse, child abuse or neglect, and must report serious injuries or deaths resulting from hunting accidents.

### Assessment & Criteria
=== "History of Present Illness"
    - **Elder Criteria:** Applies to adults 60+ or incapacitated adults 18+.
    - **Child Criteria:** Applies to minors.
    - **Hunting Accidents:** Serious bodily injury or death from hunting.

=== "Signs & Symptoms"
    - **Psychological:** Excessive passivity, fearful behavior, excessive aggression, crying, hyperactivity.
    - **Physical:** Injuries inconsistent with mechanism of injury (MOI).
    - **Neglect:** Inappropriate clothing for weather, inadequate hygiene, absence of caregivers, malnutrition.

=== "Differential Considerations"
    - **Elder/Child Abuse Types:** Physical, Sexual (or exploitation), Emotional/Psychological/Mental, Neglect, Abandonment.
    - **Hunting Reporting:** Must report within 5 days to Dept of Game and Inland Fisheries (DGIF).


### Interventions

| 2 | Assess for physical signs or injuries that are inconsistent with the reported mechanism of injury (MOI). | | • | | • | |
| 3 | Assess for signs of neglect (hygiene, clothing, malnutrition, absence of caregivers). | | • | | • | |
| 4 | **Report Elder Abuse:** Notify local DSS or Adult Protective Services (APS). If imminent danger of death/serious injury or death occurred, report to medical examiner and law enforcement. | | • | | • | |
| 5 | **Report Child Abuse:** Notify local DSS or Child Protective Services (CPS). If transporting, it is acceptable to report to the attending ED physician. | | • | | • | |
| 6 | **Report Hunting Injuries:** Notify Dept of Game and Inland Fisheries (DGIF) for serious bodily injury or death. | | • | | • | |
| 7 | Document all notifications (e.g., "made notification to Dr. [ ]", hotline calls) thoroughly on the PCR. | | • | | • | |

### Pearls
1. ⚠ **Mandated Reporter Legal Penalties:** Failure to report child abuse leads to monetary penalties (§63.2-1509). Failure to report hunting accidents is a Class 4 misdemeanor.
2. **Hotlines & Contacts:** APS 888-832-3858; DGIF 804-367-1258 or 804-367-2251.
3. **Documentation:** Ensure all physician handoffs and hotline notifications are explicitly documented on the patient care report (PCR).
