# Poquoson Fire and Rescue Patient Care Protocols
## Administrative - ALS Release to BLS

### Overview
This administrative protocol outlines the procedure for safely transferring patients who do not require Advanced Life Support (ALS) care to Basic Life Support (BLS) providers. This ensures appropriate resource utilization while maintaining a high standard of patient care and safety during the transfer of the Attendant in Charge (AIC) role.

### Assessment & Criteria
=== "History of Present Illness"
    - Complete and pertinent patient history
    - Evaluation of current complaint

=== "Signs & Symptoms"
    - Full set of baseline vital signs
    - Comprehensive physical assessment findings

=== "Differential Considerations"
    - ALS care is not currently medically necessary
    - ALS care will likely not be necessary in the near future (including pain management)
    - BLS provider capability and willingness to accept patient


### Interventions

| 1 | **Comprehensive Assessment**: Complete pertinent patient history, physical assessment, and full set of vital signs. | | | | • | |
| 2 | **Clinical Verification**: Confirm ALS care is not currently medically necessary and will likely not be necessary in the near future (to include pain management). | | | | • | |
| 3 | **BLS Verification**: Verify BLS provider is willing to accept responsibility and capable of providing a level of care that meets the assessed and documented needs. | | | | • | |

| 5 | **Medical Control**: Contact Medical Control when there is a concern of the appropriateness of the release. | | | MC | MC | MC |
| 6 | **BLS Documentation**: Complete a Patient Care Report (PCR). | | | | | |
| 7 | **ALS Documentation**: Complete documentation in, or as an addendum to, the PCR (physical findings, transfer of care, physician consultation). | | | | • | |
| 8 | **QA/QI Review**: Any ALS Release to BLS must be reviewed by the agency Quality Assurance/Quality Improvement process. | | | • | | • |
| 9 | **Incident Reporting**: Inappropriate release by QA/QI should be reported quarterly to the **Poquoson Fire and Rescue** PI Committee. | | | | • | |

### Pearls
1. ⚠ **Thorough Assessment**: The ALS provider is strictly responsible for a comprehensive clinical assessment before handoff to ensure no hidden acuity exists.
2. **Documentation Requirements**: Both providers have strict documentation responsibilities. The BLS provider completes the primary PCR, while the ALS provider must complete an addendum detailing physical findings, the transfer of care process, and any physician consultation.
3. **Medical Control**: Always consult Medical Control (**Poquoson OMD**) for edge cases or when there is any concern about the appropriateness of the release to BLS.
4. ⚠ **Pain Management**: Anticipated need for pain management is an ALS indicator; do not release patients to BLS if they will likely require analgesia in the near future.
