# Poquoson Fire and Rescue Patient Care Protocols
## Administrative - Stroke Field Triage

### Overview
To rapidly evaluate and get patients experiencing a suspected stroke to the most appropriate definitive care facility based on symptom onset time and severity.

### Assessment & Criteria
=== "History of Present Illness"
    - Time of onset / "Last seen normal" time
    - Symptom onset < 24 hours vs > 24 hours (or unknown)

=== "Signs & Symptoms"
    - **BEFAST Exam:** Altered mental status, abnormal/slurred speech, facial droop, unilateral weakness, balance instability, vision changes
    - **RACE Exam:** Facial palsy/droop, arm weakness, leg weakness

=== "Differential Considerations"
    - **Blood Glucose:** Must be > 60 mg/dL (if < 60, refer to Hypoglycemia protocol)
    - **LVO Suspicion:** RACE Score 5-9 indicates suspected Large Vessel Occlusion
    - Immediate Life-Threats (e.g., uncontrolled airway, cardiac arrest)


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | **Time Management:** Maintain a goal scene time of less than 15 minutes. | | • | | • | |
| 2 | **Diagnostics:** Assess blood glucose. If < 60 mg/dL, treat per Hypoglycemia protocol. | | • | | • | |
| 3 | **Neurological Assessment:** Evaluate patient using **BEFAST** criteria and **RACE** scale. | | • | | • | |
| 4 | **Immediate Life-Threats:** If uncontrolled airway, cardiac arrest, etc. cannot be addressed during transport, route to the nearest ED. | | • | | • | |
| 5 | **Routing (< 24 Hours & LVO):** If Symptoms < 24 Hours (or unknown) AND Suspected LVO (RACE Score 5-9), transport to closest Comprehensive Stroke Center IF it does not increase transport time by 30 mins or greater. If it does, transport to nearest Primary Stroke Center. | | • | | • | |
| 6 | **Routing (> 24 Hours OR No LVO):** If Symptoms > 24 Hours OR Low Suspicion for LVO (RACE Score 0-4), transport to closest Stroke Center (Acute Stroke Ready, Primary, or Comprehensive). | | • | | • | |
| 7 | **Stroke Alert Activation:** If RACE ≥ 1, activate stroke alert system by notifying appropriate facility early. | | • | | • | |
| 8 | **Communication:** Early notification to Medical Control and/or Stroke Center. During transport, call COR with Name and DOB for rapid registration. | | • | | • | |

### Pearls
1. ⚠ **Time is Brain:** It is critical to accurately record the "last seen normal" time to determine the appropriate treatment window.
2. ⚠ **Information Gathering:** Gather family contact info and provide it to the receiving facility for immediate consent and history needs.
3. • **Routing Exceptions:** Immediate life-threats that cannot be managed during transport supersede stroke routing guidelines; go directly to the nearest ED.
