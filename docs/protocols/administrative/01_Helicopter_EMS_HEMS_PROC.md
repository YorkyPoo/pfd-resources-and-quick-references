# Poquoson Fire and Rescue Patient Care Protocols
## Administrative - Helicopter EMS (HEMS)

### Overview
Guidelines and indications for utilizing Helicopter EMS for patient transport and specialty care.

### Assessment & Criteria
=== "History of Present Illness"
    - STEMI
    - Stroke
    - Trauma
    - Burns
    - Critical Pediatrics
    - Ventricular Assist Device (LVAD/BiVAD/VAD)

=== "Signs & Symptoms"
    - Wilderness rescue
    - Ambulance egress or access impeded at the scene by road conditions, weather, or traffic

=== "Differential Considerations"
    - The air ambulance service can provide needed medical capability at the scene


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | **Request HEMS:** Request air ambulance standby/response according to department procedure (no medical control authorization needed). | | • | | • | |
| 2 | **Establish Landing Zone:** Establish a safe and logistically feasible landing zone. | | • | | • | |
| 3 | **Provide Patient Info:** Establish number of patients, primary injuries/medical condition, and approximate patient weight(s). Forward this information to the responding air medical service. | | • | | • | |
| 4 | **Prepare Patient:** Trauma patients should be fully immobilized. **IV/IO** access should be established as time allows (within provider scope). | | • | | • | |
| 5 | **Designate Reporter:** On scene, designate one individual who will be responsible for providing the patient report(s) to the flight crew upon arrival. | | • | | • | |
| 6 | **Provide Documentation:** Ensure PCR is provided to receiving facility as per 12 VAC 5-31. | | • | | • | |

### Pearls
* ⚠ VAD patients require coordination with VAD Coordinator or Medical Control.
* HEMS requested via department procedure without online medical control authorization.
