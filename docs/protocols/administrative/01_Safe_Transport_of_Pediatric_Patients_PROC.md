# Poquoson Fire and Rescue Patient Care Protocols
## Administrative - Safe Transport of Pediatric Patients

### Overview
Best practices for the ambulance transportation of ill or injured pediatric patients to provide safe transport for all occupants. Ambulances are NOT EXEMPT from state child safety laws.

### Assessment & Criteria
=== "History of Present Illness"
    - Pediatric patient requiring transport
    - Need for medical or trauma transport

=== "Signs & Symptoms"
    - Patient weight < 40 kg (88 lbs) requires child restraint
    - Patient status: Trauma vs Medical

=== "Differential Considerations"
    - Passenger considerations (parents, caregivers, non-patients)
    - State child safety law compliance


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :---: | :---: | :---: | :---: | :---: |
| 1 | **Secure Equipment:** Secure all monitoring devices and other equipment to prevent injury. | | • | | • | |
| 2 | **Safe Driving:** Drive cautiously and at safe speeds, observing applicable traffic laws. | | • | | • | |
| 3 | **Child Restraint:** Ensure pediatric patients < 40 kg (88 lbs) are restrained with approved child restraint devices secured per the manufacturer’s instructions. | | • | | • | |
| 4 | **Provider Safety:** Ensure that all EMS providers use available restraining devices or systems during transport. | | • | | • | |
| 5 | **Caregiver Safety:** Secure parents or caregivers using safety belts. | | • | | • | |
| 6 | **Alternative Transport:** Attempt to arrange for transportation of adults and children who are not patients by alternative means. | | • | | • | |
| 7 | **Comfort Measures:** For pediatric patients having medical conditions aggravated by stress, make every reasonable attempt to maximize safety while comforting the child. | | • | | • | |

### Pearls
• ⚠ NEVER hold, attempt to hold, or permit parents or caregivers to hold the pediatric patient during transport.
• ⚠ Ambulances are NOT EXEMPT from state child safety laws. DO NOT allow parents, caregivers or any passenger to be unrestrained during transport.
• ⚠ Infants and children meeting Trauma Triage criteria should be transported in a child restraint device unless requiring relocation to or placement on an immobilization device.
