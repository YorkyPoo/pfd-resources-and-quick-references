# Poquoson Fire and Rescue Patient Care Protocols
## Administrative Policies - Patient Restraint

### Overview
Guidelines and procedures for physical and chemical patient restraint to ensure the safety of patients and responders. Patient restraint should be utilized only if the patient is exhibiting behavior that is an immediate danger to self or others.

### Assessment & Criteria
=== "History of Present Illness"
    - Patient is exhibiting behavior that is an immediate danger to self or others
    - Determine minimum level of physical restraint required
    - Evaluate personnel needed to safely restrain patient

=== "Signs & Symptoms"
    - **Hypoxia**
    - **Hypoglycemia**
    - **Head injury**
    - Signs of drug or alcohol use
    - Unsteady gait, slurred speech
    - Other signs of potential mental incapacity

=== "Differential Considerations"
    - Manner of violence (record comments verbatim)
    - Why provider felt threatened
    - Concern for outcome without interventions
    - Could patient be treated without restraints?
    - Law enforcement present/accompanied patient to ED
    - Kind and location of restraints used
    - Frequency of respiratory/mental status change assessments


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | **Request Law Enforcement if needed**<br>Call for assistance prior to attempting restraint procedures. | | • | | • | |
| 2 | **Rule out/treat medical causes**<br>Consider and treat medical causes of combativeness (hypoxia, head injury, hypoglycemia). | | • | | • | |
| 3 | **Place patient supine on backboard**<br>**NEVER PRONE**. | | • | | • | |
| 4 | **Secure extremities/check circulation every 15 mins**<br>Attempt lower extremities first (soft restraints around both ankles), then arms at sides. Secure backboard to stretcher. If necessary, utilize cervical spine precautions. | | • | | • | |
| 5 | **Apply SpO2 and monitor respiratory status**<br>Evaluate respiratory/cardiac status to assure no airway compromise exists; apply supplemental oxygen if indicated. | | • | | • | |
| 6 | **Avoid impeding respiratory function with chest straps**<br>DO NOT tighten chest straps to the point of impeding respiratory function. | | • | | • | |
| 7 | **Chemical Restraint: Assess vitals <5 mins then q10 mins**<br>And prior to any repeat dose. Monitor ECG, obtain 12-Lead, consider IN medications and establish IV/IO if possible. | | | | • | |
| 8 | **Refer to Behavioral Emergencies Protocol for sedatives**<br>Sedative agents may be needed to restrain violently combative patients. (Refer to Poquoson Protocol(s) for Behavioral Emergencies). | | | | • | |

### Pearls
* ⚠ **NEVER restrain a patient prone.**
* ⚠ **Restraint should be minimum level necessary** to accomplish patient care and ensure safe transportation (soft restraints may be sufficient).
* ⚠ **Ensure restraints do not impede evaluation or respiratory function.**
* Avoid placing restraints in such a way as to prevent evaluation of the patient’s medical needs.
* Constant evaluation of your patient’s airway status is extremely important.
* Do not endanger yourself, other responders, or the patient.
* Document reason for lack of vital signs assessment if unable to obtain them.
