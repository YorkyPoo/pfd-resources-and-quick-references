# Poquoson Fire and Rescue Patient Care Protocols
## General - Heat Exposure (Hyperthermia)

### Overview
Guidelines for managing heat-related illnesses, distinguishing between heat exhaustion and heat stroke, and initiating rapid cooling and fluid resuscitation.

### Assessment & Criteria
=== "History of Present Illness"
    - Extreme heat exposure

=== "Signs & Symptoms"
    - CNS depression
    - Arrhythmias
    - Hot/dry vs normal skin

=== "Differential Considerations"
    - Heat stroke vs exhaustion
    - Hypovolemia


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Place patient in cool environment | | • | | • | |
| 2 | Remove excess clothing | | | • | | • |
| 3 | Normal skin temp: Apply cool compresses | | | • | | • |
| 4 | Hot skin temp (Heat Stroke): Treat as life-threatening/start rapid cooling (cool mist, ice packs to armpits/groin/neck, wet sheet) | | | • | | • |
| 5 | Shivering: Slow the cooling process | | | • | | • |
| 6 | Suspected heat stroke/hypovolemia: **0.9% Normal Saline 20 mL/kg up to 1000 mL bolus** IV/IO | | | | • | |
| 7 | Continuously reassess patient | | | • | | • |

### Pearls
1. ⚠ Hot skin to touch indicates a life-threatening emergency (Heat Stroke) - begin rapid cooling immediately.
2. ⚠ If patient begins to shiver, slow the cooling process as shivering generates heat.
3. ⚠ Continuously reassess fluid needs during resuscitation.
