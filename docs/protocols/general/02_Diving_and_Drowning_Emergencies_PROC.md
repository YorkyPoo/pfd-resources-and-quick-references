# Poquoson Fire and Rescue Patient Care Protocols
## General - Diving and Drowning Emergencies

### Overview
Guidelines for managing acute submersion, non-fatal drowning, and decompression injuries, emphasizing airway management, rewarming, and specialized transport.

### Assessment & Criteria
=== "History of Present Illness"
    - Submersion
    - Compressed air diving

=== "Signs & Symptoms"
    - Altered mental status (AMS)
    - Seizures
    - Vision changes
    - Respiratory distress
    - Hemoptysis
    - Chest pain
    - Headache
    - Focal neurologic signs
    - Ataxia
    - Pain in joints / extremities

=== "Differential Considerations"
    - Hypothermia
    - Pulmonary barotrauma
    - Arterial gas embolism
    - Decompression sickness


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Follow Universal Patient Care / Initial Patient Contact protocol. Remove the patient from the water. | | • | | • | |
| 2 | Consider spinal motion restriction. Refer to Spinal Motion Restriction procedure. | | • | | • | |
| 3 | Assess airway and ventilation. Refer to Airway Management and Ventilatory Support protocol. | | • | | • | |
| 4 | Remove wet clothing and dry the skin. Keep the patient warm. | | • | | • | |
