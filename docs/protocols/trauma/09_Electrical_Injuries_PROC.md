# Poquoson Fire and Rescue Patient Care Protocols
## General - Electrical Injuries

### Overview
Guidelines for managing electrical contact and lightning injuries, emphasizing scene safety, reverse triage, and specific fluid resuscitation.

### Assessment & Criteria
=== "History of Present Illness"
    - Electrical contact
    - Lightning strike

=== "Signs & Symptoms"
    - Burns
    - Cardiac arrest
    - Arrhythmias

=== "Differential Considerations"
    - Scene safety
    - Reverse triage


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Follow Universal Patient Care / Initial Patient Contact protocol | | • | | • | |
| 2 | Ensure scene safety / patient not energized. Expose area and remove constricting items. | | • | | • | |
| 3 | Multiple patients: Treat cardiac arrest FIRST (reverse triage) | | • | | • | |
| 4 | Assess airway (manage by least invasive means necessary) | | • | | • | |
| 5 | Estimate TBSA/burn depth (2nd/3rd degree only) | | | • | | • |
| 6 | Apply clean dry dressings. Prevent loss of body heat and keep patient warm. | | | • | | • |
| 7 | Obtain 12-Lead ECG / Monitor rhythm continuously | | | • | | • |
| 8 | Establish IV/IO (x2 preferred on non-burned skin) | | | | • | |
| 9 | SBP < 90 & TBSA > 20%:<br>• Adult: **0.9% Sodium Chloride 500 mL/hr IV/IO**<br>• Ped (6-12y): **0.9% Sodium Chloride 250 mL/hr IV/IO**<br>• Ped (≤5y): **0.9% Sodium Chloride 125 mL/hr IV/IO** | | | | • | |

### Pearls
1. ⚠ Ensure scene safety and patient is not energized - DO NOT become a second victim.
2. ⚠ Lightning strikes with multiple victims require REVERSE triage (treat cardiac arrest first).
