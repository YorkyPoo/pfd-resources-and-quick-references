# Poquoson Fire and Rescue Patient Care Protocols
## General - Hemorrhage Control

Guidelines for managing life-threatening hemorrhage across various anatomical locations including extremity, junctional, and truncal injuries.

### Assessment & Criteria
=== "History of Present Illness"
    - Blunt/Penetrating trauma
    - Crush injury

=== "Signs & Symptoms"
    - Uncontrolled bleeding
    - Pelvic instability

=== "Differential Considerations"
    - Hemodialysis access bleeding
    - Airway/breathing


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Follow Universal Care / Reassess bleeding after treatments performed. | | • | | • | |
| 2 | **Head:** Apply direct pressure / Hemostatic dressing (avoid eyes/mouth). | | • | | • | |
| 3 | **Neck:** Apply occlusive dressing / direct pressure (DO NOT pack). | | • | | • | |
| 4 | **Extremity:** Apply direct pressure/packing ➔ Tourniquet 2-3" above wound ➔ 2nd Tourniquet if needed. | | • | | • | |
| 5 | **Junctional (Axilla or femoral):** Pack tightly with hemostatic gauze / hold pressure x3 mins. | | • | | • | |
| 6 | **Thoracoabdominal:** Apply direct pressure / chest seal (DO NOT pack). | | • | | • | |
| 7 | Apply pelvic binder if unstable pelvis suspected. | | • | | • | |
| 8 | Consider pain control per Poquoson Protocol(s). | | • | | • | |
| 9 | Contact Poquoson OMD / Medical Control if hemorrhage cannot be controlled and transport immediately. | MC | MC | MC | MC | MC |

### Pearls
* ⚠ Do NOT release tourniquet once applied.
* ⚠ Document date/time/location on tourniquet and report to ED.
* ⚠ Do NOT pack neck or thoracoabdominal wounds.
* ⚠ Consider pre-positioning tourniquets for crush/suspension injuries before release.
* Blood pressure cuff is an acceptable alternative if a commercial TQ is unavailable. Improvised tourniquets are unreliable.
* Hemodialysis access sites may result in life-threatening hemorrhage. Direct pressure/pressure dressing should be used first; followed by tourniquet in the setting of life-threatening hemorrhage.
* If a tourniquet is placed, an alert patient may require analgesia to manage tourniquet-associated discomfort. See Pain Control protocol.
* If extremity wound is not accessible, a tourniquet may be placed high on the extremity. Follow manufacturer’s recommendations.
