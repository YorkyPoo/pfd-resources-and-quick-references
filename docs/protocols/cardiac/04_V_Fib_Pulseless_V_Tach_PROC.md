# Poquoson Fire and Rescue Patient Care Protocols
## General - V-Fib / Pulseless V-Tach

Guidelines for the management of pulseless ventricular tachycardia and ventricular fibrillation, emphasizing high-quality CPR and rapid defibrillation.

### Assessment & Criteria
=== "History of Present Illness"
    - Unresponsive
    - Apneic
    - Pulseless

=== "Signs & Symptoms"
    - VF/VT on monitor or AED advise shock

=== "Differential Considerations"
    - Valid DDNR/Obvious death exclusions
    - Reversible causes


### Interventions

| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Begin high-quality CPR / determine DNR status | | • | | • | |
| 2 | Consider spinal motion restriction | | | • | | • |
| 3 | Establish IV/IO (Humeral IO preferred) | | | | • | |
| 4 | Fluid Resuscitation: **0.9% Sodium Chloride up to 1 L** (Adult) / **20 mL/kg** (Ped) | | | | • | |
| 5 | Defibrillation: **200 J Biphasic / 360 J Monophasic** (Adult) / **2 J/kg initial, 4 J/kg subsequent** (Ped) (AED for EMR/EMT, Manual for I/P) | | • | | • | |
| 6 | **[Epinephrine](../pharmacology/Epinephrine.md) 1 mg (0.1mg/mL) IV/IO** (Adult) / **0.01 mg/kg IV/IO** (Ped) q3-5 mins | | | | • | |
| 7 | First Antiarrhythmic: **[Amiodarone](../pharmacology/Amiodarone.md) 300 mg IV/IO** (Adult) / **5 mg/kg** (Ped max 300mg) OR **[Lidocaine](../pharmacology/Lidocaine.md) 1 mg/kg IV/IO** (Adult max 100mg, Ped max 50mg) | | | | | • |
| 8 | Second Antiarrhythmic: **[Amiodarone](../pharmacology/Amiodarone.md) 150 mg IV/IO** (Adult) / [MC] **5 mg/kg** (Ped max 150mg) OR **[Lidocaine](../pharmacology/Lidocaine.md) 0.5 mg/kg IV/IO** (Adult max 100mg) / **1 mg/kg** (Ped max 50mg) | | | | | • |
| 9 | Persistent VF/VT: **[Magnesium Sulfate](../pharmacology/Magnesium%20Sulfate.md) 2 g IV/IO** (Adult) / [MC] **40 mg/kg** (Ped max 2g) | | | | | • |
| 10 | Dialysis/Crush/Hyperkalemia: **[Calcium Chloride](../pharmacology/Calcium%20Chloride.md) 1 g IV/IO** (Adult) / **20 mg/kg** (Ped max 1g) THEN flush THEN **[Sodium Bicarbonate](../pharmacology/Sodium%20Bicarbonate.md) 50 mEq IV/IO** (Adult) / **1 mEq/kg** (Ped max 50 mEq) | | | | | • |
| 11 | Proceed to Post ROSC Care if pulses return | | • | | • | |
| 12 | Contact Poquoson OMD / Medical Control for guidance / termination of resuscitation | MC | MC | MC | MC | MC |

### Pearls
• High-quality, uninterrupted CPR is the most important intervention in cardiac arrest.
• Ensure optimal defibrillator pad placement.
• ⚠ Flush lines well between **[Calcium Chloride](../pharmacology/Calcium%20Chloride.md)** and **[Sodium Bicarbonate](../pharmacology/Sodium%20Bicarbonate.md)** to prevent precipitation.
• Reversible causes (H's and T's).
