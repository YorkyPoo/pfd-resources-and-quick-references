# Poquoson Fire and Rescue Patient Care Protocols
## Neurologic - Seizure

### Overview
Guidelines for managing witnessed or reported seizures, focusing on airway protection, preventing injury, and pharmacological intervention with benzodiazepines.

### Assessment & Criteria
=== "History of Present Illness"
    - Witnessed/reported seizure
    - Trauma
    - Overdose
    - Pregnancy

=== "Signs & Symptoms"
    - Active seizing
    - Altered mental status
    - Airway compromise

=== "Differential Considerations"
    - Exclude Eclampsia if pregnant >20 wks / <8 wks post-partum
    - Hypoglycemia


### Interventions

| 1 | Prevent injury (DO NOT restrain while seizing) | | • | | • | |

| 6 | Use HALF dose **Midazolam** for frail/hypotensive/elderly >65y | | | | • | |
| 7 | Contact Poquoson OMD / Medical Control and initiate transport if seizing >5 mins after 2nd dose of **Midazolam** | MC | MC | MC | MC | MC |

### Pearls
• ⚠ Both seizures and benzodiazepines cause significant respiratory depression - monitor airway closely.
• ⚠ Status epilepticus is a medical emergency requiring rapid treatment and transport.
• Pediatric febrile seizures require ED evaluation.
