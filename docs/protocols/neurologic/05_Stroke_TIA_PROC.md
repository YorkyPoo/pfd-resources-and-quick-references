# Poquoson Fire and Rescue Patient Care Protocols
## Neurologic - Stroke / TIA

### Overview
Guidelines for the rapid assessment and transport of patients with suspected Stroke/TIA, utilizing the BEFAST assessment and RACE scale to determine appropriate destination routing for LVOs.

### Assessment & Criteria
=== "History of Present Illness"
    - Time last seen normal
    - Anticoagulant use
    - Recent surgery/trauma

=== "Signs & Symptoms"
    - BEFAST: Balance, Eyes, Face, Arms, Speech, Time
    - Altered mental status

=== "Differential Considerations"
    - Calculate RACE scale (0-9)
    - Blood Glucose <60


### Interventions
| # | Action / Intervention | EMR | EMT | A | I | P |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Universal Care / Assess BEFAST | | • | | • | |
| 2 | Assess RACE scale for Large Vessel Occlusion (LVO) | | • | | • | |
| 3 | Record Time Last Seen Normal and family contact info | | • | | • | |
| 4 | Assess blood glucose (Treat <60 per Hypoglycemia protocol) | | | • | | • |
| 5 | Goal scene time <15 minutes / Initiate rapid transport | | | • | | • |
| 6 | Triage: Symptoms <24 hrs & RACE 5-9 -> Transport to closest Comprehensive Stroke Center (if <30 min delay) | | | • | | • |
| 7 | Triage: Symptoms >24 hrs OR RACE 0-4 -> Transport to closest Primary Stroke Center | | | • | | • |
| 8 | Establish 18g IV in Right AC (preferred for rapid imaging) | | | | • | |
| 9 | Call Poquoson COR at receiving facility with Name/DOB for rapid registration | | | • | | • |

### Pearls
• ⚠ Immediate life-threats (uncontrolled airway/arrest) go to the nearest ED.
• ⚠ Bring medications or medication list with patient.
• ⚠ Be prepared to answer ED questions regarding anticoagulants/bleeding disorders.
• RACE ≥ 5 indicates high probability of LVO requiring endovascular therapy.
