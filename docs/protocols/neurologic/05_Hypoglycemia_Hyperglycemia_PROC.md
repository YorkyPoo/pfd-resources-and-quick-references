# Poquoson Fire and Rescue Patient Care Protocols
## Neurologic - Hypoglycemia / Hyperglycemia

### Overview
Guidelines for managing symptomatic hypoglycemia and hyperglycemia, utilizing **oral glucose**, IV **[Dextrose](../../pharmacology/Dextrose.md)**, IM **[Glucagon](../../pharmacology/Glucagon.md)**, and fluid resuscitation.

### Assessment & Criteria
=== "History of Present Illness"
    - Diabetic history
    - Altered mental status

=== "Signs & Symptoms"
    - Symptomatic hyperglycemia (blood glucose greater than 300 mg/dL)
    - Symptomatic hypoglycemia (blood glucose less than 60 mg/dL)
    - Newborn hypoglycemia (less than 72 hours, blood glucose less than 40 mg/dL)

=== "Differential Considerations"
    - Ability to swallow/protect airway for **oral glucose**


### Interventions

| 3 | **Hypoglycemia (IV/IO access available)**:<br>Administer ADULT: **[Dextrose](../../pharmacology/Dextrose.md) 10% 125 mL (12.5 g) IV/IO**<br>PEDIATRIC: **[Dextrose](../../pharmacology/Dextrose.md) 10% 5 mL/kg IV/IO**. Maximum dose **125 mL (12.5 g)**.<br>*Repeat dose in 5 minutes if no response.* | | | | • | |
| 4 | **Hypoglycemia (No IV/IO access)**:<br>Administer ADULT: **[Glucagon](../../pharmacology/Glucagon.md) 1 mg IM**<br>PEDIATRIC (< 25 kg): **[Glucagon](../../pharmacology/Glucagon.md) 0.2 mg/kg IM**. Maximum dose **1 mg**.<br>*Repeat dose in 10 minutes if no response.* | | | | • | |
| 5 | **Symptomatic Hyperglycemia**:<br>Consider ADULT: **0.9% Sodium chloride up to 1 L IV/IO** in **250 – 500 mL** boluses<br>PEDIATRIC: **0.9% Sodium chloride 20 mL/kg IV/IO**.<br>*Repeat as needed to maintain adequate perfusion, up to **60 mL/kg** total.* | | | | • | |

### Pearls
1. ⚠ Ensure patient can swallow and protect airway before administering **oral glucose**.
2. **[Dextrose](../../pharmacology/Dextrose.md) 10%** is preferred over **D50** to reduce tissue necrosis risk if extravasation occurs.
